About the website

This static website was created using Pure.CSS and is hosted in Microsoft Azure. The purpose was to be able to host an 'about me' site for virtually nothing. The estimated monthly costs are ~Â£0.01.

How does it work?

I write the website as you would expect in HTML, CSS and JavaScript. But instead of paying someone else to host it, I have a storage account in Azure with a publicly readable blob containing the files for the website.

Why is it so cheap?

The main costs of a storage account are the size of the data and the number of writes, in my case the majority of the usage will come from reads, and the total size of the files will be minimal. This results in ultra-low running costs.
